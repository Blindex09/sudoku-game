HISTÓRICO DE VERSÕES
Jogo de Sudoku em Java

=== VERSÃO 1.0.0 (2025-07-28) ===

✓ FUNCIONALIDADES PRINCIPAIS IMPLEMENTADAS:

• Sistema completo de jogo de Sudoku
• Geração automática de puzzles válidos
• 4 níveis de dificuldade (Fácil, Médio, Difícil, Expert)
• Interface de console colorida e intuitiva
• Sistema de pontuação baseado em desempenho
• Validação de jogadas em tempo real
• Sistema de dicas inteligente
• Funcionalidade pausar/resumir jogo
• Opção resetar jogo para estado inicial
• Resolver automaticamente
• Estatísticas detalhadas do jogo
• Tratamento robusto de erros

✓ ARQUITETURA E DESIGN:

• Arquitetura MVC (Model-View-Controller)
• Programação orientada a objetos
• 12 classes Java bem organizadas
• Separação clara de responsabilidades
• Código documentado e comentado
• Tratamento de exceções abrangente

✓ INTERFACE E USABILIDADE:

• Interface colorida no terminal (ANSI)
• Menus numerados intuitivos
• Feedback visual para jogadas
• Destaque para células fixas vs editáveis
• Barra de progresso visual
• Estatísticas em tempo real
• Mensagens de erro claras

✓ SISTEMA DE PONTUAÇÃO:

• +10 pontos por jogada correta
• -5 pontos por erro
• -15 pontos por dica utilizada
• Bônus de tempo (até 1000 pontos)
• Penalidade baseada em erros e dicas

✓ NÍVEIS DE DIFICULDADE:

• Fácil: 40 números preenchidos inicialmente
• Médio: 30 números preenchidos inicialmente
• Difícil: 20 números preenchidos inicialmente
• Expert: 17 números preenchidos inicialmente
